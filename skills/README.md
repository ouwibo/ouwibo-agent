# Skills (SKILL.md)

This folder lets you define "skills" for Ouwibo Agent using Markdown files.

## Structure

- `skills/<skill_id>/SKILL.md`

Where `<skill_id>` must match:

- lowercase letters and numbers: `a-z`, `0-9`
- underscore or dash: `_` `-`
- max 64 characters

Examples:

- `skills/general/SKILL.md`
- `skills/research/SKILL.md`

## Tips

- Start with a single `# Title` heading.
- Put a one-line description right after the title (it will appear in the UI).
- Keep instructions practical and explicit: what to do, what to avoid, and how to format answers.
