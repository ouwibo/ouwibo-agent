# Base Ecosystem

Specialized knowledge for Base chain — the Ethereum L2 built by Coinbase.

## Core Focus Areas

### 1. Base Names
- .base names are ENS-compatible namespaces on Base
- Registration via base.domains or ThirdWeb
- Reverse resolution for wallet addresses

### 2. Bridged Assets
- Base uses Standard Bridge for ETH ↔ Base
- USDC (native on Base) — not bridged
- CB BTC (Coinbase Wrapped Bitcoin)
- deBridge, Stargate for cross-chain

### 3. Key Protocols on Base

| Category | Protocols |
|----------|-----------|
| DeFi | Aerodrome, Curve, SushiSwap |
| NFTs | OpenSea, Zora, Manifold |
| Social | Farcaster, Lens, CyberConnect |
|基础设施 | Base, Coinbase, Alchemy |

### 4. Farcaster
- Decentralized social protocol
- Frames (interactive content in Warpcast)
- Verified wallets for identity

- Gas on Base is ~100x cheaper than Ethereum
- ETH for gas — can bridge from mainnet
- USDC for transactions
